# find slope and intercept of the line corresponnding to each hideen node (1st hidden layer) 
def findSlopeIntercept(coefs,bias):
    hidden_wts=coefs[0]
    bias_wts=bias[0]
    num=len(hidden_wts[0])
    slopIntercept=[]
    for i in range(num):
        w=[bias_wts[i],hidden_wts[0,i],hidden_wts[1,i]]
        slope=w[1]/w[2]*(-1)
        intercept=w[0]/w[2]*(-1)
        slopIntercept.append([slope,intercept])
    return(slopIntercept)

Explain how the neural network decision surface changes, when increasing the number of hidden nodes from 3, to 4, to 5.

The number of hidden nodes equaled the number of lines whose overlap defines the decision surface. The resulting areas served as predictions of the class the points are. Once the number of hidden nodes was increased to 5, there was overfitting.

Explain what the benefits and risks of increasing number of hidden nodes are, respectively.

Benefits: Increased accuracy, up to a certain number of hidden nodes. Better fits more complex data. In this example, the decision boundary is not linear, so was better captured by an increase in nodes.

Risks: Increase chance of overfitting. Increase computational time and memory. Harder to explain what happens internally in the model.
